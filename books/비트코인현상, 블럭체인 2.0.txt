. "비트코인현상, 블럭체인 2.0" 책

. 화폐란 무엇인가? 

  - 금속주의자 의견 : 상품, 실제 물체가 존재해야 함.
  - 법정화폐주의자 의견 : 사회, 개인의 신용이 필요
  - 돈은 시스템적으로 설명되어야 함, 서로간에 신용이 필요함.

. 가상화폐 란?

. 전세계적으로 통용되는 화폐가 있었으면 좋겠다.

. 비트코인 

  - 나카모토 사코 ( 최초 언급한 사람, 나카모토 사코라는 사람이 실제 인물인지 모름 )
  - 1 대 1 전자 통화 시스템을 나카모토 사코씨가 암호학자들에게 제안함 
  - <정의> 디지털 서명으로 만들어진 체인 ( 원소유자는 과거 거래 내역들 뭉치에 다음번 소유자의 공개 키를 디지털 서명을 하는 것 ) 

. 시티은행 - 전자 화폐 시도한 적이 있었음. 하지만 ... ( 잘 안 되었음 )

. 가상 화폐 : 탈 중앙화

. 블럭체인 : 중앙 시스템이 없다. 정보는 각각의 Node들에 저장되어 있다. ( 분산화 )

. 비트코인의 한계

  - 비트코인은 초당 7건 처리됨 ( 느림 )
  - Visa 네트웍은 초당 1만건 처리함
  - 비트코인은 금과 달리 본질적인 가치가 없다.
  - 실리콘 밸리 이외의 지역은 잘 사용 안됨 ( 지금 현재는 어떤지 확인 필요 )
  - 일자리가 줄어든다.

. 비트코인의 미래
  - 분권화가 대세다. ( 탈 중앙화 등 )

 
* 하지만 가상화폐 시대는 반드시 온다. ( 만약 디지털 달러가 만들어진다면 어떻게 될까? )
